Playwright CI/CD Automation Framework with Jenkins & Allure
Project Overview

Built an end-to-end test automation framework using Playwright, Jenkins, GitHub, and Allure Reports. The framework executes automated UI tests through a Jenkins CI/CD pipeline and generates detailed Allure reports for test execution analysis.

Tech Stack
Playwright
JavaScript
Jenkins
GitHub
Allure Reports
Node.js
Key Features

✅ Automated UI testing with Playwright

✅ CI/CD pipeline integration using Jenkins

✅ Source code management with GitHub

✅ Allure Report integration for rich test reporting

✅ Automated test execution on every pipeline run

✅ Test execution history, screenshots, and logs in Allure Reports

Pipeline Flow

GitHub Repository → Jenkins Pipeline → Playwright Test Execution → Allure Results Generation → Allure Report Publishing

Outcome

Successfully implemented a complete CI/CD automation workflow where Playwright tests are executed automatically through Jenkins and detailed Allure reports are generated after each build.
